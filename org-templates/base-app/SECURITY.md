# Security Policy

## Reporting a vulnerability

이메일: security@rebellions.ai (PGP: ?)
72 시간 내 ack, 30 일 내 patch (T0 service 의 경우 24h ack / 7 day patch).

## Supported versions

| Version       | Supported |
|---------------|-----------|
| 최신 main      | YES       |
| 직전 minor (-1)| YES       |
| 그 이하         | NO        |

## Disclosure

CVE 공개 전 사전 통지 가능. coordinated disclosure 선호.
